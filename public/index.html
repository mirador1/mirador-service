🗼Mirador

A watchtower over industrial platform practices. Spring Boot 4 / Java 25 backend + Angular 21 frontend, demonstrating 30+ patterns you'd expect to see in a well-run production stack — on a GKE cluster that spins up on demand.

🚪 There is no public URL on purpose

The cluster does not expose any port to the public internet. No Grafana, no Argo CD, no API — all reached through kubectl port-forward tunnels from a developer laptop (see ADR-0025). It also sleeps by default to keep costs at ~€2 / month (ADR-0022).

To see the live demo: clone the repo, run bin/cluster/demo/up.sh, then bin/cluster/port-forward/prod.sh, then open the Angular UI locally and pick the prod-tunnel environment in the topbar.

What's in the demo

GitOps

Argo CD reconciles the cluster from main. Self-heal + prune enabled.

Secrets

External Secrets Operator + Google Secret Manager via Workload Identity.

Observability

OpenTelemetry → LGTM stack (Loki + Grafana + Tempo + Mimir) in-cluster.

Resilience

Resilience4J (CB + retry), Bucket4J rate limiting, idempotency filter.

Security

JWT + Keycloak OIDC, brute-force lockout, API keys, PodSecurity restricted.

Supply chain

SBOM (syft), Grype CVE scan, Dockle, hadolint, cosign keyless signing.

API versioning

Spring Framework 7 native @RequestMapping(version="2.0+").

Idempotency

IETF draft-compliant Idempotency-Key header with Redis backing.

Chaos

Chaos Mesh experiments triggered from the frontend Angular UI.
