# `.owasp-data/` — OWASP Dependency-Check local database

_This README was auto-generated by the `owasp-data-readme` Maven execution
after the first successful run of `mvn verify -Preport`. It is **not** meant
to be edited in place — the template lives at `build/owasp-data-README.md`
and is copied here by `maven-antrun-plugin` whenever the OWASP scan runs._

## Why this directory exists

The OWASP Dependency-Check plugin needs the full NVD (National Vulnerability
Database) — ~240 MB of CVE records — to scan the project's dependencies.
Without a local cache, it re-downloads the entire NVD on every scan:

- **Cold-cache first scan**: 5–15 minutes (limited by public NVD rate limits)
- **Warm-cache incremental scan**: 10–30 seconds (only the delta)

Keeping `.owasp-data/` around turns a flaky network operation into a fast
local query. The GitLab CI `owasp-dependency-check` job caches this directory
so the warm-cache speed applies across pipeline runs.

## Files

| File                                   | Role                                                                              |
| -------------------------------------- | --------------------------------------------------------------------------------- |
| `odc.mv.db`                            | H2 embedded database holding every NVD CVE record. The heart of the cache (~230 MB). |
| `jsrepository.json`                    | Retire.js vulnerability database for JavaScript libs (downloaded even with the Node analyzer disabled). |
| `jsrepository.json.properties`         | Timestamp metadata for `jsrepository.json` — plugin uses this to decide refresh.  |
| `publishedSuppressions.xml`            | Community-maintained false-positive suppressions from the dependency-check team. |
| `publishedSuppressions.xml.properties` | Timestamp metadata for `publishedSuppressions.xml`.                              |
| `cache/`                               | Plugin's scratch space during a scan. Normally empty between runs.               |
| `README.md`                            | This file (auto-generated).                                                      |

## Operations

**Rebuild from scratch** (corrupted cache):
```bash
rm -rf .owasp-data
mvn verify -Preport
```

**Update without full scan**:
```bash
mvn org.owasp:dependency-check-maven:update-only
```

**NVD API key** (recommended — public API is heavily rate-limited):
```bash
mvn verify -Preport -Dnvd.api.key=<key>
```
Get a key at <https://nvd.nist.gov/developers/request-an-api-key>.
